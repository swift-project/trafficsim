Detailed Description

Contains functions and variables for the simple implementation of the voice library. This is most suitable for a pilot client.

Note
Due to the internal nature of this class, it is only valid to have one(1) instance of it at any given time

typedef void(* Cvatlib_Voice_Simple::onHardwareDeviceReceivedEvent)(Cvatlib_Voice_Simple *obj, const char *name, void *cbVar)

A callback for when an audio device is received

Parameters
[in]objPasses in a reference to the Cvatlib_Voice_Simple object to which it is registered
[in]namePasses in the name of the device
[in]cbVarPasses in a variable defined when the callback is installed
Note
When operating on audio devices the 0-based index is passed in. This callback does not pass in the index, so in order to get the index the client must somehow save the exact order in which each instance of this callback is received
This callback will not indicate if it it passing in an input or output device, the program should only have one user of this callback running at a time, separate callback functions, or use cbVar to distinguish the calls
This callback does not indicate when the last device has been received
If saving devices for user settings it is highly recommended to save by device name and not index since indexes can change but device names usually do not

enum Cvatlib_Voice_Simple::agc_enum

Automatic Gain Control Responses

Results from the overall mic tests after doing a squelch test followed by a mic test

Note
These are bit fields so more than one error can be given at a time
Enumerator
agc_Ok 

The test went ok

agc_BkgndNoiseLoud 

The overall background noise is very loud and may be a nuisance to others

agc_TalkDrownedOut 

The overall background noise is loud enough that others probably wont be able to distinguish speech from it

agc_TalkMicHot 

The overall mic volume is too hot, the user should lower the volume in the windows volume control panel

agc_TalkMicCold 

The overall mic volume is too cold, the user should raise the volume in the windows control panel and enable mic boost if needed
